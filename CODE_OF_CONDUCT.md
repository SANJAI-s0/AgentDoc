# Code of Conduct

## Our Commitment

We are committed to making participation in this project a respectful, safe, and constructive experience for everyone, regardless of age, body size, disability, ethnicity, gender identity, gender expression, experience level, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Expected Behavior

Contributors and participants are expected to:
- be respectful and professional
- assume positive intent and communicate clearly
- give constructive feedback
- welcome questions and different perspectives
- focus on solving problems collaboratively
- respect project boundaries, decisions, and review processes

## Unacceptable Behavior

Examples of unacceptable behavior include:
- harassment, discrimination, or hateful conduct
- insults, personal attacks, or intimidation
- trolling or deliberately disruptive behavior
- publishing private information without permission
- repeated bad-faith arguments or hostile communication
- any conduct that creates an unsafe environment

## Scope

This Code of Conduct applies in:
- repository discussions
- pull requests
- issue threads
- project documentation and review communication
- any project-related community space

## Reporting

If you experience or witness unacceptable behavior, report it to the project maintainers through the repository communication channel or the designated maintainer contact for this project.

When reporting, include:
- what happened
- where it happened
- when it happened
- any helpful context or evidence

## Enforcement

Project maintainers may take any action they consider appropriate in response to unacceptable behavior, including:
- warning the contributor
- removing comments or content
- restricting participation
- temporary or permanent ban from project spaces

## Maintainer Responsibility

Maintainers are responsible for clarifying and enforcing this Code of Conduct fairly and consistently.
