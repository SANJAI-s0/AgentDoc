# Sample Documents

This folder contains ready-to-use demo samples for the document types exposed in the frontend upload form.

Included categories:
- `invoice`
- `receipt`
- `kyc_form`
- `loan_application`
- `insurance_claim`
- `shipping_document`
- `contract`
- `handwritten_form`

Each category contains:
- `sample_input.txt`: realistic plain-text payload for testing upload and extraction logic
- `expected_output.json`: structured reference output with expected core fields

These files are intentionally simple and readable for demos, prompt tuning, extraction validation, and UI walkthroughs.
