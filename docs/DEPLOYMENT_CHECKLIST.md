# AgentDoc Deployment Checklist

## Pre-Deployment

### 1. Accounts Setup
- [ ] GitHub account created
- [ ] Render account created (free tier)
- [ ] MongoDB Atlas account created (free M0 cluster)
- [ ] Google Gemini API key obtained

### 2. MongoDB Atlas Configuration
- [ ] Free M0 cluster created
- [ ] Database user created with password
- [ ] Network access configured (0.0.0.0/0 whitelisted)
- [ ] Connection string copied
- [ ] Database name set to `agentdoc`

### 3. Repository Setup
- [ ] Repository forked/cloned to GitHub
- [ ] All files committed and pushed
- [ ] `render.yaml` present in root
- [ ] `.gitignore` configured properly

## Render Deployment

### 4. Blueprint Deployment
- [ ] Connected GitHub repository to Render
- [ ] Created new Blueprint
- [ ] Render detected `render.yaml`
- [ ] Three services created:
  - [ ] agentdoc-landing (Static Site)
  - [ ] agentdoc-backend (Web Service)
  - [ ] agentdoc-frontend (Web Service)

### 5. Backend Environment Variables
- [ ] `GEMINI_API_KEY` - Your Gemini API key
- [ ] `MONGODB_URI` - MongoDB Atlas connection string
- [ ] `MONGODB_DB_NAME` - Set to `agentdoc`
- [ ] `DJANGO_SECRET_KEY` - Auto-generated by Render
- [ ] `DJANGO_DEBUG` - Set to `0`
- [ ] `DJANGO_ALLOWED_HOSTS` - Backend URL from Render
- [ ] `DJANGO_CORS_ALLOWED_ORIGINS` - Frontend URL from Render
- [ ] `DJANGO_CSRF_TRUSTED_ORIGINS` - Frontend URL from Render
- [ ] `USE_LOCAL_STORAGE` - Set to `1`
- [ ] `DJANGO_SESSION_COOKIE_SECURE` - Set to `1`
- [ ] `DJANGO_CSRF_COOKIE_SECURE` - Set to `1`

### 6. Frontend Environment Variables
- [ ] `VITE_API_BASE` - Backend API URL (e.g., https://backend.onrender.com/api)
- [ ] `NODE_VERSION` - Set to `20.x`

### 7. Landing Page Configuration
- [ ] Static site deployed
- [ ] No environment variables needed
- [ ] Publish directory set to `landing-page`

## Troubleshooting

### Common Issues

#### Backend Won't Start
- [ ] Check MongoDB connection string
- [ ] Verify Gemini API key is valid
- [ ] Check environment variables
- [ ] Review build logs
- [ ] Ensure Python 3.11+

#### Frontend Can't Connect
- [ ] Verify CORS settings
- [ ] Check VITE_API_BASE URL
- [ ] Ensure backend is running
- [ ] Check browser console for errors

#### Build Timeout
- [ ] Remove unused dependencies
- [ ] Check build command
- [ ] Verify requirements.txt
- [ ] Consider paid tier

#### Database Connection Failed
- [ ] Whitelist 0.0.0.0/0 in MongoDB
- [ ] Check connection string format
- [ ] Verify database user credentials
- [ ] Test connection locally

## Notes

**Free Tier Limitations:**
- Services sleep after 15 minutes inactivity
- Cold start takes 30-60 seconds
- Local storage is ephemeral
- 15 minutes max build time

**Keep-Alive Solution:**
Use UptimeRobot to ping services every 14 minutes to prevent sleep.
