## 📖 Usage Guide

### For Local Development

1. **Copy template**:
   ```bash
   cp .env.example .env
   ```

2. **Update required values**:
   ```env
   GEMINI_API_KEY=your_key_here
   MONGODB_URI=mongodb://localhost:27017
   DJANGO_SECRET_KEY=generate-random-string
   ```

3. **Start development**:
   ```bash
   python manage.py migrate
   python manage.py seed_demo
   python manage.py runserver
   ```

### For Render Deployment

1. **Don't use .env file** - Use Render environment variables

2. **Set in Render dashboard**:
   - GEMINI_API_KEY
   - MONGODB_URI
   - DJANGO_ALLOWED_HOSTS
   - DJANGO_CORS_ALLOWED_ORIGINS
   - USE_LOCAL_STORAGE=1

3. **Render auto-generates**:
   - DJANGO_SECRET_KEY

## 🆘 Common Issues

### Issue: Missing Required Variables

**Error**: `KeyError: 'GEMINI_API_KEY'`

**Solution**: Set required variables
```env
GEMINI_API_KEY=your_key_here
MONGODB_URI=mongodb://localhost:27017
```

### Issue: CORS Errors

**Error**: CORS policy blocked

**Solution**: Update CORS settings
```env
DJANGO_CORS_ALLOWED_ORIGINS=http://localhost:5173
DJANGO_CSRF_TRUSTED_ORIGINS=http://localhost:5173
```

### Issue: MongoDB Connection Failed

**Error**: Connection refused

**Solution**: Check MongoDB URI
```env
# Local
MONGODB_URI=mongodb://localhost:27017

# Atlas
MONGODB_URI=mongodb+srv://user:pass@cluster.mongodb.net/
```
